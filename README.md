# ChoromeExtension
Chrome拡張機能


#用語

- Browser Action(ツールバーに常駐)
- Page Action(特定のページで起動)
- Content Script(特定のページで自動実行)
- Context Menu(右クリックメニュー)

ファイルの説明

manifest.json(これがプログラムの関係性を記述するための仕組み)
